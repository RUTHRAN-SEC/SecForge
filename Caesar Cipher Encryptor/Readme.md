# Caesar Cipher Encryptor

## Project Description

This project is a simple Python based Caesar Cipher Encryptor.
It allows the user to enter text to Encrypte and asks shift value to Encrypted message.
The Caesar Cipher is one of the oldest encryption techniques, used historically by Julius Caesar.
This project demonstrates basic cryptography concepts, string manipulation, and algorithm implementation using Python.
It is designed for learning purposes and cybersecurity fundamentals.

## How To Run

- Make sure Python 3 is installed.
- Download or clone this repository.
- Open terminal inside the project folder.

### Run the program:
```
python caesar_encryptor.py
```

It asks to Enter:

- Enter the text to encrypt
- Enter the shift you want

Example

Enter a input:

- Enter the text to encrypt: HELLO
- Enter the shift you want: 3

Output:
```
HELLO
```
